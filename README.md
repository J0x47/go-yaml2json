Convert YAML to JSON.

## Installation
```bash
go get -u gopkg.in/peter-edge/yaml2json.v1
```

## Import
```go
import (
    "gopkg.in/peter-edge/yaml2json.v1"
)
```

Inspired by https://github.com/bronze1man/yaml2json, a command line tool with
the same effect.
